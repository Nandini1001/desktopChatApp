# Django Real-Time Chat Application
A lightweight, real-time web chat application built with the Django framework. This project allows users to create or join chat rooms and communicate instantly without page refreshes using AJAX polling.
# Features
Dynamic Room Creation: Users can join existing rooms or create new ones simply by entering a room name.
Real-Time Messaging: Uses jQuery and AJAX to fetch and display new messages every second.
User Identification: Simple username-based entry for quick access to chats.
Persistent Storage: All chat history and room details are stored in a local SQLite database.
Responsive UI: Styled with a modern gradient background and clean container-based layout.
# Tech Stack
Backend: Python 3.x, Django 4.0.6
Frontend: HTML5, CSS3, JavaScript (jQuery/AJAX)
Database: SQLite
